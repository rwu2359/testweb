Wow, this is a cool way to provide help within FileMaker.

It can be updated on-the-fly and be directly available within my FileMaker solution.

FileMaker Server also runs a web server and can host html files. Using something like the following, your solution can pull from your FileMaker Server

Substitute ( Get ( FilePath ) ; [ "fmnet:/" ; "http://" ] ; [ Get ( FileName ) & ".fmp12" ; "path/to/server/hosted/file.html" ] )
